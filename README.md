# Python Practice Programs

##  Description
This repository contains a collection of independent Python programs
written for practice while learning Python fundamentals.
Each file focuses on a specific concept such as loops, functions,
recursion, and basic games.

##  Topics Covered
- Star pattern programs
- Snake Water Gun game
- Functions and return values
- Basic recursion programs
- Conditional statements and loops
